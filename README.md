# Linux-Manager
Simple scripts to help manage VPS.

> All scripts are only tested in Ubuntu noble until now.

## [Init]

The `init.sh` scripts will install common but essential packages in the VPS, we highly recommand you to run `init.sh` once before u running the others scripts provided by Linux-Manager.

```shell
sudo ./init.sh
```
